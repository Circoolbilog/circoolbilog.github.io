# Installing Tools
Create a Conda Environment
```shelll
#Create the Conda Environtment
conda create -n bioinfo
#Activate the newly created environment
source activate bioinfo
```
Install trinotate using mamba
```shelll
mamba install bioconda::trinotate
```
